Roadmap

Feature Complete
Use Javascript to invoke Xamarin Forms app
Get Javascript invocation working with a local html file
Get Javascript invocation working on this GitHub Pages hosted site
Get Javascript invocation working on an external site (such as Google)
Create a payload object to pass from javascript to my app that contains plenty of information about the invocation event
